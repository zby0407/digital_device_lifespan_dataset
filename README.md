# Digital device Lifespan Dataset

This dataset provides battery life and specification data for select flagship smartphones and laptops from 2024-2025.

Mobile data sourced from Geekerwan [enter](https://www.socpk.com/), Xinping Tech [enter](https://xpkjpk.com/), and WHYLAB [enter](https://space.bilibili.com/8969156/lists/157?type=season).

Laptop data sourced from Geekerwan [enter](https://www.socpk.com/) and Laptop Bar Reviews [enter](https://space.bilibili.com/367877?spm_id_from=333.337.0.0).

Some technical data is sourced from the official websites of Snapdragon, Intel, AMD, NVIDIA, and various OEM manufacturers.
